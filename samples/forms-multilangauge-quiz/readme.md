# 🌐 Multilanguage Quiz in Microsoft Forms Possible 📝

./assets/demo.png

## Summary
This guide is designed to help you create a Multilanguage quiz in Microsoft Forms. Since Microsoft Forms does not natively support multiple languages in a single form, you will need to manually write the questions in each language including answers. This can be a daunting and time-consuming task, but since you have Copilot the process becomes much more easier.

## Prompt💡

Create a quiz on a specyfic topic: [topic]. 
The number of questions should be [number]

Make this quiz in several langauages: [languages] and make sure that questions and answers are given in those languages. 
Use parameterss below:
topic: physics
number: 3
difficulty level: easy
languages: English, Polish, Portugeese
